
# Online E-Learning Platform

It is a full-featured online learning platform that allows students to browse and purchase courses, read topic-focused blogs, and access a built-in digital library with downloadable PDF books.  
The platform is built using **HTML, CSS, and PHP**, focusing on simplicity, accessibility, and clean user experience.

---

##  Features

### **1. Course Marketplace**
- Browse structured courses
- Detailed course pages with overview and curriculum
- Add-to-cart / purchase flow (PHP-based backend logic)
- User-friendly course viewing interface

### **2. Blog & Learning Articles**
- Blog section covering various topics
- Dynamic article loading
- Reader-friendly UI layout

### **3. Digital Library**
- Organized collection of academic books
- In-platform reading (embedded PDF viewer)
- Direct download option for offline study

### **4. Authentication System**
- Secure user login & registration
- Session-based access control (PHP)

### **5. Clean UI/UX**
- HTML/CSS responsive layout
- Simple and distraction-free learning environment

---

##  Tech Stack

**Frontend:**  
- HTML5  
- CSS3  

**Backend:**  
- PHP (Core PHP)
